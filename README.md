# upload-to-leon README

将图片上传到狮子座插件, 在setting中设置狮子座相关参数后，点击鼠标右键会显示上传图片按钮，选择图片可直接在编辑器中写入上传后的链接

在setting的Extensions可设置上传狮子座到服务器的所需参数，输入后即可使用插件进行快捷的上传图片并写入图片链接
